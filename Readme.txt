В приложении 3 метода:
1) Добавление данных с использованием JSON строки в качествет параметра https://localhost:7283Entity/AddEntity?JsonStr={"id":"cfaa0d3f-7fea-4423-9f69-ebff826e2f89","operationDate":"2019-04-02T13:10:20.0263632+03:00","amount":23.05} 
2) Получение объекта по ID https://localhost:7283/Entity/GetEntity?id=047c9f0c-7d76-4ab3-8d86-34b636f9f25c
3) Получение списка всех объектов https://localhost:7283/Entity/GetEntitys

Замечания:
1) В методе AddEntity происходит десирилизация строки JSON. Если возникает ошибка десирилизация, в таком случаи получаем сообщение, что серверу не удается сохранить представление, необходимое для выполнения запроса.
2) В методе GetEntity. Если объект по ID не найден, то получаем сообщение о том что объект не найден

Хранение данных.
В программе реализован интерфейс IReposity для реализации хранения сущностей. Так же была добавлена зависимость интерфейса IRepository c помощью метода для AddSingleton.
